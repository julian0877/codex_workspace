# 钢梁板式吊耳验算桌面程序

## 启动方式

双击 `启动吊耳验算程序.bat`。

如果没有反应，请确认 Windows 已安装 Python 3.12 或兼容版本，并且 `pythonw` 在系统 PATH 中。

## 功能

- 选择 5t、10t、15t、30t、40t、50t 模板。
- 修改吊装荷载、吊点数量、每吊点耳板片数、角度、材料、耳板尺寸、孔径、焊缝参数。
- 自动计算耳板承载力和双面角焊缝承载力。
- 同时显示吊点钢丝绳拉力 T3 和单片耳板分担设计力。
- 显示每项验算的计算值、允许值、利用率和结论。
- 导出 Word 格式计算书（`.docx`）。

## 说明

当前版本按现有 Excel 表格的验算链整理，适合作为 Windows 桌面原型使用。导出的 Word 计算书包含输入参数、主要中间量、验算结果和总体结论。正式工程应用前，建议结合采用规范版本、企业校审口径和原始 Excel 逐项复核。
